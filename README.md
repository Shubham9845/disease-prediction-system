# 🧬 BioPredict — Disease Prediction & Healthcare Analytics System

> ML-powered disease prediction system combining Biotechnology domain knowledge with Data Analytics and Python Web Development.

## 🌐 Live Demo
👉 [Click here to try the app](https://biopredict-2026.streamlit.app)

## 📌 Project Overview
BioPredict is a full-stack machine learning web application that predicts diseases based on symptoms selected by the user. Built using real Kaggle medical data, it combines Biotechnology domain expertise with Data Analytics and Computational Biology.

This project serves as:
- ✅ B.Tech Final Year Project
- ✅ Data Analyst Portfolio Project
- ✅ Computational Biology Demonstration
- ✅ Python Web Development Showcase

## 🚀 Features
- 🔬 **Disease Predictor** — Select symptoms and get instant ML prediction with confidence %
- 📊 **EDA Dashboard** — 6+ interactive Plotly charts on real medical data
- 🤖 **ML Model Insights** — Feature importance, confusion matrix, cross validation scores
- 📋 **Patient Records** — SQLite database with Excel export
- 🏠 **Home Page** — Disease cards with risk levels (High/Medium/Low)
- ⬇️ **Export** — Download EDA report and patient records as Excel
